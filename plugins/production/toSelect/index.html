$().toSelect(options)

Replace an input field with a select field

Example

Options

  • id: The ID of the generated SELECT
  • class: The class of the generated SELECT
  • options: (required) An array containing the information about the OPTIONs that will be generated. Each entry should have the following format:
    • text: The text to be displayed for an OPTION.
    • value: The actual value of an OPTION that will be stored in the input field
    • selected: (true/false) determines if the current OPTION should be preselected.
